# SAO Gold Edition – MT4 & MT5 Expert Advisors

A collection of MetaTrader 4 and MetaTrader 5 Expert Advisors for XAUUSD (Gold),
built for educational, research, and backtesting purposes.

## Versions
- MT4 – SAO4 Gold Edition
- MT5 – SAO5 Gold Edition V1
- MT5 – SAO5 Gold Edition V2 (Dynamic Risk)

## Features
- Range-based trading logic
- ATR volatility filter
- EMA trend filter
- Risk-based lot sizing
- Optional partial close
- Dynamic risk reduction (V2)
